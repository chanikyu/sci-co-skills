## 📦 스킬

| 스킬 | 기능 |
|---|---|
| 🧬 [amplicon-analysis](https://github.com/chanikyu/SciCo-Skills/wiki/amplicon-analysis) | 16S/ITS 마이크로바이옴 파이프라인 — 알파 & 베타 다양성(PCoA, PERMANOVA)과 차등 존재비, 저널 그림 포함. |
| 📊 [scientific-data-viz](https://github.com/chanikyu/SciCo-Skills/wiki/scientific-data-viz) | 실제 데이터로 만드는 논문 품질의 저널 그림 — 정확한 수치, 20종 팔레트, PERMANOVA 포함 통계. |
| 🧫 [scientific-workflow-viz](https://github.com/chanikyu/SciCo-Skills/wiki/scientific-workflow-viz) | BioRender 스타일의 개념도 이미지 프롬프트, 선택적으로 Google Nano Banana(Gemini) 렌더링 지원. |

## 🚀 설치

```bash
/plugin marketplace add chanikyu/SciCo-Skills
/plugin install SciCo-Skills
```

전체 설정과 스킬별 가이드는 **[Wiki](https://github.com/chanikyu/SciCo-Skills/wiki)**에 있습니다
(`amplicon-analysis`는 Python ≤ 3.12가 필요합니다).
