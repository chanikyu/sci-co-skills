## 📦 Skills

| Skill | 功能说明 |
|---|---|
| 🧬 [amplicon-analysis](https://github.com/chanikyu/SciCo-Skills/wiki/amplicon-analysis) | 16S/ITS 微生物组流程——alpha 与 beta 多样性（PCoA、PERMANOVA）以及差异丰度，并配套期刊级图表。 |
| 📊 [scientific-data-viz](https://github.com/chanikyu/SciCo-Skills/wiki/scientific-data-viz) | 从真实数据生成可发表的期刊级图表——数值精确、20 种配色方案、内含 PERMANOVA 等统计检验。 |
| 🧫 [scientific-workflow-viz](https://github.com/chanikyu/SciCo-Skills/wiki/scientific-workflow-viz) | BioRender 风格的概念图图像提示词，可选通过 Google Nano Banana（Gemini）直接渲染成图。 |

## 🚀 安装

```bash
/plugin marketplace add chanikyu/SciCo-Skills
/plugin install SciCo-Skills
```

完整安装说明与各 skill 的指南都在 **[Wiki](https://github.com/chanikyu/SciCo-Skills/wiki)**
上（`amplicon-analysis` 需要 Python ≤ 3.12）。
